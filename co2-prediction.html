What did I do?

Using pandas and scikit-learn I did a model that predict the C02 emissions using as independent variables the electricity usage (kWh) and the lagging current reactive power (kVarh). I also did the pre-processing work using the scikit-learn KNNImputer. This small model get a mean absolute error of 0.0019.
For this project I used a dataset called Steel Industry Energy Consumption that is from the DAEWOO Steel Co. Ltd in Gwangyang, South Korea.
