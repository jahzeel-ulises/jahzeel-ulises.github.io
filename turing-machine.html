Turing Machine

A Turing machine is a device that manipulates symbols on a strip of tape according to a table of rules. Despite its simplicity, a Turing machine can be adapted to simulate the logic of any computer algorithm and is particularly useful in explaining the functions of a CPU within a computer. (ref)

What did I do?

I did a simple desktop application with python that let you write your own programs with a simple rule language, and then shows you an animation of the program running on the strip.
